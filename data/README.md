# Данные

В проекте использовался датасет RuSentiTweet.

Для эксперимента были оставлены три класса:
- negative
- neutral
- positive

Классы `speech` и `skip` были исключены на этапе предобработки, так как не относятся напрямую к полярности тональности.

В работе использовались:
- исходные train/test файлы датасета;
- очищенные версии с тремя классами.
